I'm starting this blog to journal all the interesting things I learn everyday in my job as a 
software developer. I've decided to build my own blog engine as an excuse to try new technologies. 
I know what you're thinking, that I'm re-inventing the wheel, and you're absolutely right! Is this a wise thing to do, considering I could just install Wordpress in a few clicks, select a snazzy theme and start blogging nearly immediately? On the tech side, I'm basically a .NET guy, but I like what the Alt.NET community stands for, and I also want to dabble with new technologies, maybe Ruby and Python to get a feel for what it's like to build stuff in those environments.
